#### 通货膨胀率（通胀率）

GDP平减指数 = 名义GDP／ 实际GDP

CPI消费物价指数 = 所消费物品的平均价格

通货膨胀率 = （名义GDP - 实际GDP）／实际GDP



GDP平减指数：经济中所生产的最终物品的平均价格。

CPI消费物价指数：经济中所消费的物品的平均价格。

两者可能不同是因为GDP中的某些物品不是卖给消费者而是卖给企业或其他国家，消费者所购买的物品也可能是其他国家进口的。
